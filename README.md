Descripción

Este portfolio está desarrollado con HTML, CSS y JavaScript vanilla, con un enfoque en diseño visual, interacción y experiencia de usuario.

La web utiliza una estética dark + neón, con efectos de glassmorphism, animaciones al scroll y navegación fluida.
Cada proyecto cuenta con una sección propia donde se muestran las imágenes en gran formato mediante un visor horizontal con flechas de navegación.

Incluye:

Hero con título animado tipo máquina de escribir y glow neón.

Aparición progresiva de elementos según el scroll.

Sección de proyectos con tarjetas interactivas.

Viewer de imágenes en gran formato para cada proyecto (fotografía, branding e ilustración).

Botón “back to top” con scroll suave.

Diseño responsive adaptado a móvil, tablet y escritorio.

Animaciones y efectos

Animación de texto tipo typewriter en el HERO.

Glow neón dinámico en títulos y elementos interactivos.

Animaciones de entrada activadas por scroll (Intersection Observer).

Hover con efecto de “bombeo” en las tarjetas de proyectos.

Viewer de imágenes con transición suave y navegación por flechas.

Scroll suave con compensación del header fijo.

Indicador de progreso de scroll lateral.

Parallax sutil en secciones de proyectos.

Botón “volver arriba” con animación fluida.

Instalación

Clona o descarga este repositorio.

Abre index.html en un navegador moderno.

Asegúrate de que la carpeta assets esté completa para que estilos, scripts e imágenes funcionen correctamente.

No requiere instalación adicional ni dependencias externas.
